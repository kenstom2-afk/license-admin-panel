API Documentation

Server API (For Shell Scripts)

All server API requests require the X-Server-Key header.

# Example shell script usage:
SERVER_KEY="your_server_key_here"
BASE_URL="https://your-domain.com/api/server"

# Create a license
curl -X POST "$BASE_URL/licenses/create" \
  -H "X-Server-Key: $SERVER_KEY" \
  -H "Content-Type: application/json" \
  -d '{"product": "MyApp", "days": 30, "owner": "John Doe"}'

# Get license details
curl -X GET "$BASE_URL/licenses/LICENSE-KEY-HERE" \
  -H "X-Server-Key: $SERVER_KEY"

# Reset a license
curl -X POST "$BASE_URL/licenses/LICENSE-KEY-HERE/reset" \
  -H "X-Server-Key: $SERVER_KEY"

# Extend a license
curl -X POST "$BASE_URL/licenses/LICENSE-KEY-HERE/extend" \
  -H "X-Server-Key: $SERVER_KEY" \
  -H "Content-Type: application/json" \
  -d '{"days": 30}'

Client Verification API

All client API requests require the X-API-Key header.

# Verify a license (Python example)
import requests

api_key = "your_api_key_here"
license_key = "LICENSE-KEY-HERE"
hwid = "DEVICE-HWID-HERE"

response = requests.post(
    "https://your-domain.com/api/v1/verify",
    headers={
        "X-API-Key": api_key,
        "Content-Type": "application/json"
    },
    json={
        "license_key": license_key,
        "hwid": hwid,
        "device_name": "User's Device"
    }
)

if response.json().get("success"):
    print("✅ License is valid!")
    print(f"Product: {response.json()['license']['product']}")
    print(f"Days left: {response.json()['license']['days_left']}")
else:
    print("❌ License verification failed")
    print(f"Error: {response.json()['message']}")

Shell Script Examples

Client Verification Script

Save as verify.sh

#!/bin/bash
API_KEY="your_api_key_here"
API_URL="https://your-domain.com/api/v1/verify"

# Get HWID (Linux/Mac)
HWID=$(cat /etc/machine-id 2>/dev/null || echo "$(hostname)-$(date +%s)")

LICENSE_KEY="$1"
[ -z "$LICENSE_KEY" ] && echo "Usage: $0 LICENSE_KEY" && exit 1

response=$(curl -s -X POST "$API_URL" \\
  -H "X-API-Key: $API_KEY" \\
  -H "Content-Type: application/json" \\
  -d "{\\"license_key\\": \\"$LICENSE_KEY\\", \\"hwid\\": \\"$HWID\\"}")

echo "$response" | python3 -c "
import sys, json
data = json.load(sys.stdin)
if data.get('success'):
    print('✅ License is valid!')
    print(f'Product: {data[\"license\"][\"product\"]}')
    print(f'Days left: {data[\"license\"][\"days_left\"]}')
    sys.exit(0)
else:
    print('❌ License verification failed')
    print(f'Error: {data[\"message\"]}')
    sys.exit(1)
"
Server Management Script

Save as license-manager.sh

#!/bin/bash
SERVER_KEY="your_server_key_here"
BASE_URL="https://your-domain.com/api/server"

case "$1" in
    create)
        curl -s -X POST "$BASE_URL/licenses/create" \\
          -H "X-Server-Key: $SERVER_KEY" \\
          -H "Content-Type: application/json" \\
          -d "{\\"product\\": \\"$2\\", \\"days\\": ${3:-30}, \\"owner\\": \\"${4:-}\\"}" | python3 -m json.tool
        ;;
    get)
        curl -s -X GET "$BASE_URL/licenses/$2" \\
          -H "X-Server-Key: $SERVER_KEY" | python3 -m json.tool
        ;;
    reset)
        curl -s -X POST "$BASE_URL/licenses/$2/reset" \\
          -H "X-Server-Key: $SERVER_KEY" | python3 -m json.tool
        ;;
    extend)
        curl -s -X POST "$BASE_URL/licenses/$2/extend" \\
          -H "X-Server-Key: $SERVER_KEY" \\
          -H "Content-Type: application/json" \\
          -d "{\\"days\\": ${3:-30}}" | python3 -m json.tool
        ;;
    *)
        echo "Usage: $0 {create|get|reset|extend} [args]"
        echo "Examples:"
        echo "  $0 create MyProduct 30 \"John Doe\""
        echo "  $0 get LICENSE-KEY"
        echo "  $0 reset LICENSE-KEY"
        echo "  $0 extend LICENSE-KEY 30"
        exit 1
        ;;
esac
